# MyBatis.NET.SqlServer v1.0.0 - Release Summary

## Release Info

- Package: `MyBatis.NET.SqlServer`
- Version: `1.0.0`
- Release date: `2026-03-11`

## Highlights

- XML-based SQL mapper runtime for .NET
- Dynamic SQL tags: `<if>`, `<where>`, `<set>`, `<choose>`, `<foreach>`, `<trim>`
- Mandatory `returnSingle` contract for `<select>` statements
- Mapper proxy execution via `SqlSession`
- SQL and parameter logging support
- Async query/execute APIs
- Result mapper cache optimization with performance statistics
- Interface generation tool support via `mybatis-gen`

## Notes

- This package line starts from `v1.0.0` and will evolve incrementally by roadmap.
- Legacy package naming under `MyBatis.NET.SqlMapper` should be considered deprecated for new installs.
